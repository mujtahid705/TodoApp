#TodoApp

This is a simple to-do list app that will work as your helping hand to help you organize your work. Using this app, you can Add your tasks, Edit them if you make any mistakes, and Delete them if you don't need them. You can also sort the list By Name, Date, and Time. You can filter the list based on your priority.
